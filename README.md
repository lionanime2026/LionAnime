# 🎬 LionAnime

Egy modern anime streaming platform bejelentkezéssel, regisztrációval és admin panellel.

## 🚀 Főbb Funkciók

- 📺 Anime streaming
- 🔐 Bejelentkezés/Regisztráció
- 🎮 Admin panel (anime feltöltés, szerkesztés, törlés)
- 🎯 Könnyű videó lejátszó
- 💬 Discord integráció
- 🎨 Modern UI/UX

## 📁 Projekt Struktúra

```
LionAnime/
├── frontend/          # React frontend
├── backend/           # Node.js/Express backend
├── database/          # Database setup
└── docs/              # Dokumentáció
```

## 🛠️ Telepítés

### Frontend
```bash
cd frontend
npm install
npm start
```

### Backend
```bash
cd backend
npm install
npm start
```
